LendingClub Loan Data Analysis

In this project, I screened loan data from LendingClub to determine whether or not a given borrower would pay back the loan in full. Analyzing simple univariate and bivariate data or borrower information’s including the loan purpose, FICO score, DTI and interest rates, I employed Python and data analysis tools such as Pandas, NumPy and Scikit-learn to build the models.
Few important findings observed from this analysis were; a large percentage of the consumers used the loan for debt consolidation, there was a noticeable difference between the consumers who had high FICO and the interest rates which they were charged but there was hardly any relation between debt to income ratio and FICO score. Furthermore, other graphical forms such as the loan purpose distributions, FICO score histograms, and the interest rate vs. risk box plots enabled the identification of borrowers characteristics and interaction of variables.

Skin Cancer Analysis

PThis project aims at creating a network of Machine Learning for early diagnosis and prognosis of skin cancer based on CNNs and LSTM. Since image processing and temporal data analysis algorithms are used to create the model, skin lesions can be accurately detected and their development forecasted.
Some of the highlights of the model are CNNs for image feature extraction from dermoscopy images and LSTM for modelling patient’s history to capture the temporal information. The model was trained on three categories of split HAM10000 data set with 94% accuracy in different categories of skin cancer. This system not only enhances the accuracy of diagnosis but also becomes an easy tool to work with in the current healthcare systems hence avoiding the use of invasive means in the diagnoses if ailments hence enabling early diagnosis with high accuracy.
